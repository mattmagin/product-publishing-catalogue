# Context

## Glossary

- **Monorepo**: A single repository that contains multiple deployable applications and shared project-level tooling.
- **Workspace app**: A deployable application managed by the pnpm workspace.
- **Backend API**: The Rails API-only backend microservice in `apps/api`.
- **Frontend microfrontend**: The React TypeScript Vite application in `apps/frontend`, exposed as a Module Federation remote.
