# Context

## Glossary

- **Monorepo**: A single repository that contains multiple deployable applications and shared project-level tooling.
- **Workspace app**: A deployable application managed by the pnpm workspace.
- **Backend API**: The future Rails API-only backend microservice in `apps/api`.
- **Frontend microfrontend**: The future React TypeScript Vite application in `apps/web`, intended to be pluggable through Module Federation.
